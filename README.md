# RC eStore

A React-based e-commerce demo application built to showcase frontend architecture, state management, and client-side persistence.

## Tech Stack

- React (Vite)
- React Router
- Redux Toolkit & React Redux
- Tailwind CSS
- Firebase (client SDK)
- React Toastify for notifications
- Lucide icons

## Features

- Product listing and cart flow
- Global state management using Redux Toolkit
- Client-side routing
- Toast notifications for user feedback
- Responsive UI with Tailwind CSS

## Getting Started

```bash
git clone https://github.com/Abhishekwmg/rc-estore
cd rc-estore
npm install
npm run dev
```
